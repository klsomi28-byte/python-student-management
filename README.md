# Student Management System (Python)

## Description
The Student Management System is a Python-based application designed to manage student records efficiently. It allows users to add, view, update, and delete student information. This project helps beginners understand basic Python concepts and data management.

## Features
- Add student records  
- View student details  
- Update student information  
- Delete student records  
- Simple menu-driven interface  

## Technologies Used
- Python  

## Objective
The main objective of this project is to practice Python programming and understand how a basic management system works.

## How to Run
1. Make sure Python is installed on your system.
2. Run the Python file using:
python main.py
